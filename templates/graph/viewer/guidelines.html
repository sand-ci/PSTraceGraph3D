Guidelines

Building query

There are seven filters available to build query. Setting one of them effects the others.

The first one is Datetime range filter. Using icons you can set up values for datetime range: from and to.

The others filters are dealing with server names and IPs values. These filters are Source, Source site, Source host, Destination, Destination site, Destination host.

Each of them has relevant box where selected values are stored and dynamic table where items can be selected. Every dynamic table has fuzzy search bar at the top of it. To delete items from box just click on them.

To make all filters be updated click Apply button after any changes performed.

Toolbar

Toolbar is located at the top of the application window.

It has pagination buttons backward and forward , button to refresh graph , button to show/hide panels and button to get guidelines .

Paths visualization

Source hosts, sources, hops, missing hops, destinations and destination hosts are represented as spheres of different colors. Source hosts' and Destination hosts' spheres are larger then the others as they declare begging and end of any Path.

Paths datatable

Paths datatable is located at the bottom of application window and comprises further headers: Source, Source site, Source host, IPv6, Timestamp, Destination, Destination site, Destination host, Incomplete, Path ID.

Paths datatable has relevant fuzzy search input and shows items with pagination by ten. Clicking on item will highlight path on graph and fill relevant info in the Path info window on the right. If Path fold up in the Path datatable toolbar is active all paths of the same hash value will be highlighted, Path info window on the right will show mean data.

Also there is slider for Distance power adjustment in the Path datatable toolbar Please keep in mind that it doesn't expand space between nodes but increase power of links between them.
